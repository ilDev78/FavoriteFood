# FavoriteFood (ร้านโปรดของฉัน) 🍽️

เว็บไซต์สั่งอาหารนี้ขึ้นมาเพื่อให้กับร้านอาหารร้านหนึ่งที่ฉันชอบไปกินบ่อยๆ เขาเปิดมานานแล้วเขาไม่ได้รู้เรื่องเทคโนโลยีมากนักฉันเลยอยากที่จะทำเว็บไซต์ให้เขาเพื่อจะได้มีรายได้เพิ่มขึ้นจากทางทางอื่นเพิ่มขึ้น และฉันยังได้ความรู้จากการทำโปรเจคนี้ด้วย

## ฟีเจอร์หลัก ✨
- **หน้าเมนูอาหาร:** แสดงรายการอาหารน่าทาน พร้อมแบ่งหมวดหมู่ชัดเจน (จานหลัก, ซุป, แกง) ให้ค้นหาได้ง่าย
- **ระบบตะกร้าสินค้า:** ตะกร้าสินค้าจะจดจำรายการที่คุณเลือกไว้ในเครื่อง (Local Storage)
- **สรุปยอดอัตโนมัติ:** ระบบจะช่วยคำนวณราคารวมและค่าจัดส่งให้คุณเห็นแบบเรียลไทม์

## วิธีการใช้งาน 💡
1. เปิดเข้าไปที่หน้าแรกของเว็บไซต์เพื่อดูเมนูทั้งหมด
2. เจอเมนูที่ถูกใจ ให้กดปุ่ม **(+)** เพื่อเพิ่มอาหารลงในตะกร้า
3. เมื่อเลือกเสร็จแล้ว ให้กดที่ปุ่ม **"🛒 ตะกร้า"** มุมขวาบน
4. ในหน้าตะกร้า คุณสามารถเพิ่ม/ลดจำนวน หรือลบอาหารออกได้ตามต้องการ
5. ตรวจสอบยอดเงินให้เรียบร้อยแล้วกดปุ่ม **"ชำระเงิน"** ได้เลย

## เทคโนโลยีเบื้องหลัง 🛠️
โปรเจกต์นี้สร้างขึ้นด้วยเทคโนโลยีที่เน้นความรวดเร็วและสวยงาม:
- **EJS & HTML:** สำหรับสร้างโครงสร้างหน้าเว็บ
- **Tailwind CSS:** สำหรับตกแต่งความสวยงามของหน้าจอ
- **Vanilla JavaScript:** ใช้จัดการระบบตะกร้าสินค้าโดยไม่พึ่งพาระบบที่ซับซ้อน

## วิธีรันโปรเจกต์
```bash
#ติดตั้ง express
npm install express

#รันโปรเจกต์
npm start
```

PORT=3000
